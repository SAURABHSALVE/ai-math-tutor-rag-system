# Solution Templates for Common JEE Problem Types

## Template: Solving Quadratic Equations
1. Write equation in standard form: ax^2 + bx + c = 0
2. Calculate discriminant: D = b^2 - 4ac
3. Apply quadratic formula: x = (-b +/- sqrt(D)) / (2a)
4. Simplify and check if roots satisfy original equation
5. If D < 0, state complex roots if required

## Template: Optimization Problems (Calculus)
1. Define the objective function f(x) to maximize/minimize
2. Identify constraints and domain
3. Find f'(x) and set f'(x) = 0
4. Solve for critical points
5. Use second derivative test: f''(x) > 0 (min), f''(x) < 0 (max)
6. Check boundary values if domain is closed interval
7. State the maximum/minimum value

## Template: Probability Problems
1. Identify the sample space and total outcomes
2. Identify favorable outcomes
3. Check if events are independent/dependent
4. Apply appropriate formula:
   - Direct counting: P = favorable/total
   - Conditional: P(A|B) = P(A and B)/P(B)
   - Bayes: P(A|B) = P(B|A)*P(A)/P(B)
5. Simplify and verify 0 <= P <= 1

## Template: Limit Evaluation
1. Try direct substitution first
2. If indeterminate (0/0 or inf/inf):
   - Try factoring and canceling
   - Try L'Hopital's rule
   - Try known standard limits
   - Try rationalization (for sqrt expressions)
3. Verify by approaching from both sides if needed

## Template: Matrix/Determinant Problems
1. Write out the matrix clearly
2. For determinant: expand along row/column with most zeros
3. For inverse: check det != 0, then use formula or row reduction
4. For systems: set up augmented matrix and row reduce
5. Verify solution by substitution

## Template: Sequence and Series
1. Identify if AP, GP, or other pattern
2. Find the general term
3. Apply appropriate sum formula
4. For convergence (GP): check |r| < 1
5. Verify with first few terms

## Template: Derivative Applications
1. Identify what rate/quantity is being asked
2. Set up the function relationship
3. Differentiate using appropriate rules
4. Substitute given values
5. Interpret the result in context

## Verification Checklist
- Does the answer have correct units/dimensions?
- Does the answer make physical/mathematical sense?
- Does it satisfy the original equation/constraints?
- Are there edge cases to check?
- Is the answer simplified to lowest terms?
