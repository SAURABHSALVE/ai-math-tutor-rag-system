# Logarithms and Exponents

## Exponent Rules
- a^m * a^n = a^(m+n)
- a^m / a^n = a^(m-n)
- (a^m)^n = a^(mn)
- (ab)^n = a^n * b^n
- a^0 = 1 (a != 0)
- a^(-n) = 1/a^n
- a^(1/n) = nth root of a

## Logarithm Definition
- log_a(x) = y means a^y = x
- Domain: x > 0, base a > 0, a != 1

## Logarithm Properties
- log_a(xy) = log_a(x) + log_a(y)
- log_a(x/y) = log_a(x) - log_a(y)
- log_a(x^n) = n * log_a(x)
- log_a(1) = 0
- log_a(a) = 1
- log_a(x) = 1 / log_x(a)

## Change of Base
- log_a(x) = log_b(x) / log_b(a) = ln(x) / ln(a)

## Natural and Common Logarithms
- ln(x) = log_e(x), where e = 2.71828...
- log(x) = log_10(x) (common log)

## Solving Exponential Equations
- If a^f(x) = a^g(x), then f(x) = g(x) (same base)
- Take log of both sides: f(x) * ln(a) = g(x) * ln(b)

## Solving Logarithmic Equations
- Convert to exponential form
- Check domain: argument must be positive
- Verify solutions don't make the argument negative or zero

## Common JEE Tricks
- log_a(b) * log_b(c) = log_a(c) (chain rule)
- If log_a(x) = log_a(y) then x = y
- log_a(x) > 0 when (a > 1 and x > 1) or (0 < a < 1 and 0 < x < 1)
- log_a(x) < 0 when (a > 1 and 0 < x < 1) or (0 < a < 1 and x > 1)
