# JEE Problem-Solving Strategies

## General Strategy
1. Read the problem carefully - identify what is given and what is asked
2. Classify the topic and sub-topic
3. Draw a diagram if applicable
4. Write down known quantities and unknowns
5. Choose appropriate formula or technique
6. Solve step by step
7. Verify the answer

## Multiple Choice Strategies
- Elimination: rule out obviously wrong options
- Substitution: plug in answer choices to verify
- Dimensional analysis: check if answer has right dimensions
- Extreme cases: test boundary values
- Approximation: quick estimate to narrow down options

## Common Algebraic Approaches
- Substitution: introduce new variable to simplify
- Factoring: look for common patterns
- Symmetry: exploit symmetric expressions
- Inequality chaining: use AM-GM, Cauchy-Schwarz

## Calculus Problem Approaches
- For limits: try substitution first, then L'Hopital, then series expansion
- For maxima/minima: derivative = 0, check second derivative or boundary
- For integration: try substitution, then parts, then partial fractions

## Probability Approaches
- Count carefully: use permutations vs combinations appropriately
- Complement method: P(at least one) = 1 - P(none)
- Conditional: break into cases using Bayes' theorem
- Tree diagrams for sequential events

## Linear Algebra Approaches
- For systems: check determinant first to know if unique solution exists
- Row reduction for large systems
- Cramer's rule for small systems (2x2, 3x3)
- Eigenvalue problems: characteristic equation det(A - lambda*I) = 0

## Time Management
- Easy problems: 2-3 minutes
- Medium problems: 4-5 minutes
- Hard problems: 6-8 minutes
- If stuck for more than 3 minutes, move on and return later
