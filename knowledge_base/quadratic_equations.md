# Quadratic Equations - Detailed Guide

## Standard Form
ax^2 + bx + c = 0, where a != 0

## Solving Methods

### 1. Factoring
- Express ax^2 + bx + c as product of two binomials
- Example: x^2 - 5x + 6 = (x-2)(x-3) = 0, so x = 2 or x = 3

### 2. Quadratic Formula
- x = (-b +/- sqrt(b^2 - 4ac)) / (2a)
- Always works for any quadratic

### 3. Completing the Square
- Move constant to other side
- Add (b/2a)^2 to both sides
- Factor the perfect square trinomial

## Nature of Roots (Discriminant D = b^2 - 4ac)
- D > 0: Two distinct real roots
- D = 0: One repeated real root (x = -b/2a)
- D < 0: Two complex conjugate roots

## Relation Between Roots and Coefficients
If alpha and beta are roots of ax^2 + bx + c = 0:
- alpha + beta = -b/a (sum of roots)
- alpha * beta = c/a (product of roots)
- |alpha - beta| = sqrt(D) / |a|

## Forming Equations from Roots
If roots are alpha and beta:
- x^2 - (alpha + beta)x + (alpha * beta) = 0

## Common JEE Tricks
- If roots are reciprocals: c = a
- If roots are equal in magnitude, opposite in sign: b = 0
- If one root is zero: c = 0
- For real roots: D >= 0 gives range of parameter values
