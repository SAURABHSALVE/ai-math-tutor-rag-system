# Domain and Range Constraints

## Common Domain Restrictions
- 1/x: x != 0
- sqrt(x): x >= 0
- ln(x): x > 0
- log_a(x): x > 0, a > 0, a != 1
- tan(x): x != (2k+1)*pi/2
- sec(x): x != (2k+1)*pi/2
- sin^(-1)(x): -1 <= x <= 1
- cos^(-1)(x): -1 <= x <= 1
- 1/sqrt(x): x > 0

## Range of Standard Functions
- sin(x), cos(x): [-1, 1]
- tan(x): (-inf, inf)
- e^x: (0, inf)
- ln(x): (-inf, inf)
- x^2: [0, inf)
- |x|: [0, inf)
- sin^(-1)(x): [-pi/2, pi/2]
- cos^(-1)(x): [0, pi]
- tan^(-1)(x): (-pi/2, pi/2)

## Units and Dimensional Checks
- Angles in calculus formulas: always radians unless stated
- Probability: 0 <= P <= 1
- Lengths, areas, volumes: non-negative
- Variance: always >= 0
- Standard deviation: always >= 0

## Validity Checks
- Matrix multiplication A*B: columns of A must equal rows of B
- Inverse exists only if det != 0
- Division by zero: always check denominators
- Even roots of negative numbers: not real
- Logarithm of non-positive: undefined in reals

## Physical Reasonableness
- Speed/distance: non-negative
- Time: non-negative
- Count of objects: non-negative integer
- Probability: between 0 and 1 inclusive
